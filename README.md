🚀 Getting Started
Prerequisites

Node.js (v16 or higher)
npm

Installation
bash# Clone the repo
git clone https://github.com/rounakm20/my-portfolio.git
cd my-portfolio

# Install dependencies
npm install

# Start dev server
npm run dev
Open http://localhost:5173 in your browser.
Build for Production
bashnpm run build
